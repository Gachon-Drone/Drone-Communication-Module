# 03. Network Flow Diagram (Description)

This diagram illustrates:

- Drone internal components (PX4 + Companion Computer)
- LTE router providing outbound network path
- Public server forwarding telemetry and commands
- Web client interacting with the entire system

Arrows indicate:
- Telemetry flow (Drone → Server → Client)
- Command flow (Client → Server → Drone)

The diagram is used in documentation and presentations to explain the entire communication pipeline.
